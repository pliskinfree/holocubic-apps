Holo PC Monitor

把电脑的实时状态，浓缩进一块精致的 320×240 仪表盘。Your live PC status, distilled into a polished 320×240 dashboard.

功能亮点Highlights

01实时硬件数据Live hardware data

通过 AIDA64 RemoteSensor 显示 CPU、GPU、内存、温度、频率与风扇状态。Shows CPU, GPU, memory, temperatures, clocks and fans through AIDA64 RemoteSensor.

02三种仪表盘布局Three dashboard layouts

经典、四卡与蜂窝节点布局随时切换，兼顾信息密度与个性化观感。Switch among classic, four-card and honeycomb layouts.

03天气、时间与趋势Weather, time & trends

顶部集中显示本地天气、时间和日期，曲线持续呈现关键指标走势。Local weather, time and date sit above live history charts for key metrics.

04网页配置Web configuration

可配置数据地址、布局、CPU/GPU 名称与蜂窝强调色，无需修改设备端代码。Configure the data source, layout, hardware names and honeycomb accent color.
