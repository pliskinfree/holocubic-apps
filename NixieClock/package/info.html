Nixie Clock

用电子管风格数字显示当前时间。

Shows the current time with nixie tube digits.

功能说明

Features

  • 1电子管数字使用本地数字素材渲染复古辉光效果。
  • 2时间同步支持网络时间同步并带重试机制。修改ntp服务器为阿里云
  • 3专注显示主界面只保留时间，适合作为桌面时钟。
  • 1Tube digitsUses local digit assets for a vintage glow.
  • 2Time syncSupports network time sync with retry.
  • 3Focused displayKeeps the screen focused on time.
