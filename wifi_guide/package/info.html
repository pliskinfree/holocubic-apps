首次联网引导

WiFi Setting Guide

为 HoloCubic 提供清晰的首次配网步骤，并在联网后展示设备控制入口。

功能说明

  • 1等待配网

    显示设备热点、配网地址和分步操作说明，引导手机或电脑连接设备。

  • 2自动识别联网状态

    持续检测 Station IP，联网成功后自动切换到网络信息界面。

  • 3实时网络信息

    展示当前 WiFi、RSSI 信号强度、设备 IP 和本地域名。

  • 4离线生成二维码

    由 Lua 本地编码控制地址并通过 LVGL Canvas 绘制，无需二维码接口或外部服务。

  • 5中英文界面

    读取 Web 控制页保存的语言设置；简体中文显示中文，其他情况默认英文。

使用方法

  1. 1连接设备热点

    使用手机或电脑连接设备屏幕上显示的热点。

  2. 2完成 WiFi 配置

    等待配网页自动弹出，或打开屏幕所示的配网地址。

  3. 3进入控制网页

    联网成功后，扫描二维码，或使用设备 IP / clocteck-cubic.local 访问控制页。

应用信息

版本号2.0.1
屏幕尺寸320 × 240
安装包大小约 530 KB
设备控制域名clocteck-cubic.local

所有二维码均在设备本地生成。网络状态只从设备自身的系统接口读取，不会上传 WiFi 名称、IP 或信号数据。

项目地址：github.com/clocteck/holocubic-apps

蓝牙手柄控制
